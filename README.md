# Rate-Limiter - návrh a implementace (CZ)

## Zadání

Cílem je navrhnout a implementovat Rate Limiter pro REST API službu, jež bude omezovat počet požadavků na klienta v daném časovém intervalu. Součastí zadaní je návrh pro Single-node i Multi-node prostředí.

# Rate-Limiter - design and implementation (ENG)

## Task

The goal is to design and implement Rate Limiter for REST API service, which will be limiting number of requests per client in given time. Part of the task is to design it in away to be correct either in Single-node or Multi-node enviroment.
